# ImaginaryCraft（异想工艺）

## 模块定位

ImaginaryCraft 项目的**主模块/伞模块**，作为最终打包入口，聚合所有子模块（GoldenBoughsLib, ResonatorCombatFramework,
EGOCurios, EGOEquipment, Abnormalities）。目前处于早期开发阶段，大部分包为支架。

## 项目结构

```
src/main/
├── java/architecture/imaginarycraft/
│   ├── package-info.java               — 包描述
│   ├── core/ImaginaryCraftConstants    — 主模块常量
│   ├── datagen/                        — 数据生成
│   │   ├── Datagen                     — 数据生成入口
│   │   ├── ModDatagenBlockState        — 方块状态（空实现）
│   │   ├── ModDatagenDatapackBuiltinEntries — 数据包内置条目（空实现）
│   │   ├── ModDatagenItemModel         — 物品模型（空实现）
│   │   ├── ModDatagenParticle          — 粒子描述（空实现）
│   │   ├── ModDatagenSoundDefinitionsProvider — 音效定义（空实现）
│   │   └── i18n/ModZhCn               — 中文本地化
│   ├── init/ModSoundEvents             — 音效事件（空实现）
│   ├── util/WorldUtil                  — 世界工具（空实现）
│   ├── client/                         — （空支架：eventexecute, events, model, particle, renderer/block/effect/entity/layer/providers）
│   ├── common/                         — （空支架：behavior, menu, world/effect/item/armor/ego/weapon）
│   ├── core/registry/                  — （空支架）
│   ├── events/                         — （空支架）
│   ├── mixed/                          — （空支架）
│   └── mixin/                          — （空支架，mixins.json 已配置）
│
└── kotlin/architecture/imaginarycraft/
    └── core/                           — ImaginaryCraft.kt (@Mod), ImaginaryCraftClient.kt
```

## 模块聚合关系

```
ImaginaryCraft（伞模块）
├── GoldenBoughsLib          — 核心库（API, 工具, mixin 接口）
├── ResonatorCombatFramework — 战斗框架 + 玩家动画系统
├── EGOCurios                — EGO 饰品（Curios 槽位）
├── EGOEquipment             — EGO 武器 + 护甲
└── Abnormalities            — 异想体 + 考验实体
```

## 开发状态

当前处于**早期支架阶段**：

- 2 个 Kotlin 入口文件，13 个 Java 文件（主要是 datagen 工具）
- 大量目录已创建但为空（`client/`, `common/`, `events/`, `mixin/`, `mixed/`）
- datagen 提供程序方法均为空实现
- `imaginarycraft.mixins.json` 已配置但 mixin 列表为空
